# Attendee Management

This guide covers everything you need to know about managing attendees for events in OSPO Events Manager.

## Overview

The Attendee Management feature allows you to track and manage event participants, including speakers, attendees, and their roles in various events. This helps facilitate better collaboration and communication.

## Key Features

### Track Event Participants
- View all attendees for a specific event
- See attendee contact information
- Track attendee roles (speaker, attendee, organizer, etc.)
- Monitor attendance status

### Manage Attendee Information
- Add new attendees to events
- Update attendee details
- Remove attendees when needed
- Export attendee lists

### Role Management
- Assign roles to attendees
- Track speaker assignments
- Manage organizer permissions
- View role-based statistics

## Adding Attendees

### From the Events Page
1. Navigate to the Events page
2. Click on an event to view details
3. Go to the Attendees tab
4. Click "Add Attendee"
5. Fill in the attendee information:
   - Name
   - Email address
   - Role (Speaker, Attendee, Organizer, etc.)
   - Company/Organization (optional)
   - Notes (optional)

### From the Attendees Page
1. Navigate to the Attendees page from the main menu
2. Click "Add Attendee"
3. Select the event
4. Fill in attendee information
5. Save

## Attendee Roles

The system supports several attendee roles:

- **Speaker**: Presenting at the event
- **Attendee**: Regular participant
- **Organizer**: Event organizer or staff
- **Sponsor**: Sponsoring organization representative
- **Volunteer**: Event volunteer
- **Media**: Press or media representative

## Viewing Attendees

### By Event
- Navigate to an event's detail page
- Click on the "Attendees" tab
- View all attendees for that event
- Filter by role or search by name

### All Attendees
- Go to the Attendees page
- View all attendees across all events
- Filter by event, role, or search
- Export attendee lists

## Best Practices

### Data Entry
- Use consistent naming conventions
- Include accurate email addresses
- Add company/organization information
- Keep notes updated

### Role Assignment
- Assign roles accurately
- Update roles as needed
- Track speaker assignments separately
- Maintain organizer lists

### Communication
- Use attendee lists for event communications
- Export lists for external tools
- Keep contact information current
- Respect privacy preferences

## Integration with Other Features

### CFP Submissions
- Speakers are automatically linked to CFP submissions
- View speaker submissions from attendee details
- Track submission status per speaker

### Events
- Attendees are associated with specific events
- View attendee counts on event cards
- Filter events by attendee participation

### Assets
- Link assets to specific attendees
- Upload speaker headshots
- Attach presentation materials

## Troubleshooting

### Can't Add an Attendee
- Verify you have permission to edit the event
- Check that required fields are filled
- Ensure the event exists and is accessible

### Missing Attendee Information
- Check if attendee was added to correct event
- Verify filters aren't hiding the attendee
- Refresh the page

### Duplicate Attendees
- Use search to find existing attendees
- Merge duplicate entries if needed
- Contact admin for bulk cleanup
